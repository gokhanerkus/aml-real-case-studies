# Case 02 — Banking Network & Iran-Linked Funds Flow

## Summary
This case analyzes a financial network involving a Turkish bank, related corporate entities, and cross-border transactions linked to an Iran-connected sanctions evasion structure.

The case focuses on identifying AML risks through financial flows, ownership structures, and related-party exposure.

---

## Key Focus Areas
- Sanctions exposure  
- Related-party risk  
- Banking transaction flows  
- Cross-border financial movement  

---

## Why This Case Matters
This case demonstrates how financial institutions can be used as intermediaries in complex money laundering and sanctions evasion schemes.

It highlights the importance of:
- Detecting concentration risk  
- Identifying hidden relationships  
- Monitoring cross-border flows
