# Case 01 — Iran Sanctions Evasion Network

## Summary
This case analyzes a cross-border sanctions evasion network involving Iranian, Turkish, and UAE-linked entities.

---

## Key Focus Areas
- Sanctions evasion  
- Network-based money laundering  
- Trade-based money laundering (TBML)  
- Cross-border financial flows
  
---

## Key Insight
Early investigative analysis identified structural links within the network **before they became widely exposed through international reporting**.
