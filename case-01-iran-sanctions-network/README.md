# Case 01 — Iran Sanctions Evasion Network

## Summary
This case analyzes a cross-border sanctions evasion network involving Iranian, Turkish, and UAE-linked entities.

The case is based on early-stage open-source intelligence (OSINT) signals identified before major international investigations.
