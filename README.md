# AML Case Studies – OSINT & Financial Crime Analysis

This repository contains structured AML-style case studies based on real-world investigative work.

## Focus Areas
- Client onboarding risk assessment
- KYC / KYB and due diligence
- Source of Funds (SoF) / Source of Wealth (SoW)
- Adverse media and reputational risk
- Cross-border and high-risk jurisdiction analysis
- Escalation decision-making

## Approach
All cases are:
- Based on open-source intelligence (OSINT)
- Structured for risk assessment and decision-making
- Rewritten in a neutral, analytical format

## Note
Some cases are derived from investigative work later referenced in academic literature on illicit networks and shadow economies.
